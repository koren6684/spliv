# spliv: Patterned Sensitivity Analysis for IV

Matrix-based patterned sensitivity analysis for plausibly exogenous IV
inference with scalable fixed-effect residualization,
researcher-specified direct-effect patterns, sensitivity paths over
delta, and optional confirmatory BPE diagnostics.
